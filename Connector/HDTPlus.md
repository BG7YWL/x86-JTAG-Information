# AMD 20-pin HDT+
## Specifications
### Pin Pitch
1.27mm
### Number Of Pins
20 (2 * 10)
### Compatible Receptacles
- Samtec ASP-137098-05
### Pinout
| Name | Pin Number | Pin Number | Name |
| - | - | - | - |
| VDDIO | 1 | 2 | TCK |
| GND | 3 | 4 | TMS |
| GND | 5 | 6 | TDI |
| GND | 7 | 8 | TDO |
| TRST_L | 9 | 10 | PWROK_BUF |
| DBRDY3 | 11 | 12 | RESET_L |
| DBRDY2 | 13 | 14 | DBRDY0 |
| DBRDY1 | 15 | 16 | DBREQ_L |
| GND | 17 | 18 | TEST19 |
| VDDIO | 19 | 20 | TEST18 |
